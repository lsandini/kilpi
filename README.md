# Kilpi - Interactive Decision Support Tool for Thyroid Cancer

**Work in Progress**

An interactive web-based decision support tool for papillary and follicular thyroid cancer management, based on Finnish clinical guidelines.

## About

This is an alternative interactive format of the Duodecim clinical guideline article:

## Features

- Interactive risk stratification calculators
- Ultrasound malignancy risk assessment
- FNA (fine needle aspiration) indication calculator
- Bethesda classification management tool
- Incidental findings surveillance recommendations
- Surgical planning decision support

## Usage

Open `index.html` in a web browser. No server or build process required.
